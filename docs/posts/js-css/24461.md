# 小程序和 Web 端同构方案：kbone

**官方出品解决方案**

`kbone` 是一个致力于微信小程序和 Web 端同构的解决方案。

`kbone-ui` 是一个能同时支持 小程序(kbone) 和 vue 框架开发的多端 UI 库。

官方文档：[https://wechat-miniprogram.github.io/kbone/docs/](https://wechat-miniprogram.github.io/kbone/docs/)
