# css设置高度等于其宽度

1、使用padding-bottom的百分比值是相对包含块的宽度的知识点 ，如：

``` js 
<style type="text/css">
#container {
    width: 80%;
    height: 500px;
}
.attr {
    width: 50%;
    height: 0;
    padding-bottom: 50%;
    background-color: #008b57;
}
</style>
<div id='container'>
    <div class='attr'></div>
</div>
```

2、或使用js方法

``` js 
$(document).ready(function(){
    $('#box1').css('height',$('#box1').css('width'));
})
```
