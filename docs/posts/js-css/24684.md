# 如何查看其他小程序的AppID

**最简单方法：查看小程序资料**

1. 在微信小程序里搜索该小程序的名字，比如：“小程序示例”
2. 打开小程序，点击右上角 三个点 “...” 的按钮，出现弹层
3. 在弹层里点击 最上边小程序名字 比如：“小程序示例>”，出现小程序的资料介绍
4. 在资料介绍页面，继续点击点击右上角 三个点 “...” 的按钮，出现弹层
5. 在弹层里点击 “更多资料”

好了，点击的操作完了，现在出现了小程序的详情资料，比如：账号原始ID，AppID等等
