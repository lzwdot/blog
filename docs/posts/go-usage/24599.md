# windows下go编译成linux可执行文件

**如果编译报错：**

> cmd/go: unsupported GOOS/GOARCH pair linux /amd64
> 
> 是因为在 SET GOOS=linux 这句后面多了个空格（直接复制的命令。。。）

**先执行**：

``` js 
SET CGO_ENABLED=0
```

然后再执行

``` js 
SET GOOS=linux 
SET GOARCH=amd64
go build 
```

参考：[https://www.rootop.org/pages/4328.html](https://www.rootop.org/pages/4328.html)
