# 开源的SQL可视化工具HeidiSQL

HeidiSQL是Windows平台下的SQL可视化工具，它支持 MariaDB，MySQL，Microsoft SQL或PostgreSQL，如果想在Linux和MacOS上运行，需要使用Wine。

下载地址： [https://www.heidisql.com/download.php](https://www.heidisql.com/download.php)
