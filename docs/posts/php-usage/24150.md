# 排名前50的WP插件和主题

出于好奇心，想知道WordPress一些免费主题的排行榜，于是找到了[topfiftywp.com](https://topfiftywp.com/)这个网站， 该网站每天显示下载量排名前50的WordPress插件。它使用WordPress.org插件API从目录中托管的插件中提取前一天的下载统计信息。该网站还在右侧栏中以百分比形式显示平均评分。
