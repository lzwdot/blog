# module chunk bundle 的区别

> 详细参考：[https://bookrss.com/archives/27191/](https://bookrss.com/archives/27191/)

- module - 各个源码文件，webpack 中一切皆模块
- chunk - 多模块合并成的（打包过程中），如 entry import() splitChunk
- bundle - 最终的输出文件，也就是 asset 文件
