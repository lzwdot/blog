# Vue 和 React 的区别

相同点

- 都支持组件化
- 都是数据驱动视图
- 都使用 vdom 操作 DOM

不同点

- React 使用 JSX 拥抱 JS，Vue 使用模板拥抱 html
- React 函数式编程，Vue 声明式编程
- React 更多需要自力更生，Vue 把想要的都给你
