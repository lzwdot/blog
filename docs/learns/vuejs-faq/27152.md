异步加载，也叫懒加载。示例：

``` js
export default new VueRouter({
    routes: [
        {
            path: '/',
            component: () => import(
                /* webpackChunkName:"navigator" */
                './../components/Navigator'
            )
        },
        {
            path: '/feedback',
            component: () => import(
                /* webpackChunkName:"feedback" */
                './../components/Feedback'
                )
        }
    ]
})
```
