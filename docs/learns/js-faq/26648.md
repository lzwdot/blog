# 关于作用域和自由变量的场景题

第一个，感觉不容易

``` js 
let i
for (i = 1; i <= 3; i++) {
    setTimeout(function () {
        console.log(i) // 4 4 4
    }, 0)
}
```

第二个，简单点

``` js 
let a=100
function test(){
    console.log(a) // 100
    a =10
    console.log(a) // 10
}
test()
console.log(a) // 10
```
