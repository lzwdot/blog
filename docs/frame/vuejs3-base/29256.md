# Vite 为什么启动非常快

说一说 Vite

## Vite 是什么

- 一个前端打包工具，Vue 作者发起的项目
- 借助 Vue 的影响力，发展较快，和 webpack 竞争
- 优势：开发环境下无需打包，启动快

> 参考：https://v3.cn.vuejs.org/guide/installation.html#vite
