当有个需求，通过监听url的location.hash（即#值）变化，来实现页面刷新。如：

``` js
window.onhashchange = func;
<body onhashchange="func();">
window.addEventListener("hashchange", func, false); 
```

刷新页面：

``` js
function func(){
    location.reload();
}
```
