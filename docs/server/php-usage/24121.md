# WordPress分支ClassicPress

ClassicPress是一个社区主导的开源内容管理系统，并且是WordPress的一个分支，它将经典的TinyMCE编辑器保留为默认选项。

> [https://www.classicpress.net/](https://www.classicpress.net/)
> 
> ClassicPress官网

如果你坚强的TinyMCE编辑器拥护者， ClassicPress 或许是一个不错的选择
