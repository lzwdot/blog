# Vue-router 使用

Vue 路由模式

  • hash 模式（默认），如：http://abc.com/#/user/10在新窗口打开
  • H5 history 模式，如 http://abc.com/user/20在新窗口打开
  • 后者需要 server 端支持，因此无特殊需求可选择前者

如 nginx 配置

location / {
  try_files $uri $uri/ /index.html;
}

Vue-router 路由配置

  • 动态路由 {path:'/user/:id',component:user}
  • 懒加载 component:()=>import('./Router')
